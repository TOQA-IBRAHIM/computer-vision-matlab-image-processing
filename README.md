# Computer Vision & Image Processing (MATLAB)

## 📌 Overview
This repository contains a set of computer vision tasks implemented in MATLAB, focusing on image preprocessing and basic shape recognition.

The project demonstrates fundamental techniques used in image analysis and pattern recognition.

---

## 🎯 Objective
To apply core computer vision concepts such as image preprocessing, feature extraction, and shape detection using MATLAB.

---

## 🧠 Tasks Included
- Image preprocessing techniques (filtering, thresholding, etc.)  
- Basic image analysis and transformation  
- Shape recognition (rectangle, triangle, and other simple shapes)  

---

## 🔍 Final Project
A simple shape recognition application that identifies geometric shapes from input images using image processing techniques.


---

## 🛠️ Technologies Used
- MATLAB  
- Image Processing Toolbox  

---

## 📂 Project Files
- Multiple MATLAB scripts for different tasks and experiments  

---

## 📂 Project Type
Computer Vision – Image Processing (Academic Project)
